# Bytebeat Module

This is a local, modified copy of [greggman/html5bytebeat](https://github.com/greggman/html5bytebeat).

The original library implements a bytebeat compiler and audio processor for use in the browser. It has been forked and adapted for use in Patchies, with modifications to integrate with the Web Audio Worklet API and the Patchies node system.
