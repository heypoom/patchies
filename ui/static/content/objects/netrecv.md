Receive messages from the network.

## Usage

Type `netrecv <channelname>` to create a receiver, e.g. `netrecv chat`.

Messages from that channel will flow out of the outlet.

See the [P2P Messaging docs](/docs/network-p2p) for the room configuration, how it works and the Node.js/Bun integration.
