Speak text aloud using the browser's built-in
[Web Speech API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API).

## Setup

1. Type `tts` in the object box to create the node
2. Click the gear icon to select a voice

## Voice Selection

Choose from available system voices using the searchable dropdown. Voice
selection is persisted with the patch.

No API key required - uses your browser's native TTS engine.
