Perfect for documenting your patches, leaving notes for collaborators, or marking sections of your work.

## Features

- **Resizable**: Drag corners to resize
- **Customizable colors**: Choose from 6 preset colors (yellow, pink, blue, green, purple, orange)
- **Adjustable font size**: Small, Medium, Large, or XL
- **Double-click to edit**: Quickly enter edit mode
- **Lock**: Lock the note to prevent moving, resizing or editing.

## Usage

1. Create a `note` object
2. Double-click to edit text
3. Click the settings icon to change color and font size
4. Drag corners to resize
