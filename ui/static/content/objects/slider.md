Continuous value control with customizable range.

## Shorthand Commands

- `slider 0 100` - integer slider
- `fslider 0.0 1.0` - float slider
- `vslider` / `vfslider` - vertical variants

## Messages

- **bang** → outputs current value
- **number** → sets value and outputs
