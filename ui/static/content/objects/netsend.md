## Usage

Type `netsend <channelname>` to create a sender, e.g. `netsend chat`.

Send messages into the inlet to broadcast on that channel.

See the [P2P Messaging docs](/docs/network-p2p) for the room configuration, how it works and the Node.js/Bun integration.
