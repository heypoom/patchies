FFT (Fast Fourier Transform) analysis for frequency domain processing.

Outputs frequency bin data for audio visualization and analysis.

## Usage

See [Audio Reactivity](/docs/audio-reactivity) for FFT usage with P5.js
and Hydra, presets, and converting existing code.
