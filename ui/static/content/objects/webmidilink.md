Convert MIDI messages to [WebMIDILink](https://www.g200kg.com/en/docs/webmidilink)
format (link level 0).

Connect to [iframe](/docs/objects/iframe) to send MIDI messages to
WebMIDILink-enabled web apps.

> See [this demo](https://patchies.app/?id=x7q9819cn6zplpk) from @kijjaz
> using webmidilink with SpessaSynth.
