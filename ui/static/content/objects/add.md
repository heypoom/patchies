Add a number to incoming values.

## Usage

```
+ 1
+ 12
```

The argument sets the right operand. A number sent to the hot inlet is added to that operand
and emitted immediately.

## Inlets

- **Inlet 0** (hot): Value to transform
- **Inlet 1**: Right operand
