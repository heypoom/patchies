Transcribe speech to text using the browser's built-in
[Web Speech API](https://developer.mozilla.org/en-US/docs/Web/API/SpeechRecognition).

Click the node to toggle listening. Send `listen` to start, `stop` to finish.
Outputs transcribed text as a string message.

No API key required — uses your browser's native speech recognition engine.

## Settings

- **Language**: BCP-47 language code (e.g. `en-US`, `th-TH`, `ja-JP`)
- **Continuous**: Keep listening after each utterance
- **Interim Results**: Output partial results while speaking

## Browser Support

Works in Chrome and Edge. Firefox and Safari have limited support.
