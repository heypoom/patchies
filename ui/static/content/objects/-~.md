Subtract right signal from left signal, sample-by-sample.

## Usage

```
osc~ 440 → -~ ← osc~ 441
```

Useful for computing signal differences, differential signals,
and phase cancellation effects.

Pass a number as a creation argument to subtract a constant
(e.g. `-~ 0.5`).

## See Also

- [+~](/docs/objects/+~) - add signals
- [*~](/docs/objects/*~) - multiply signals
