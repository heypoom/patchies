Subtract right signal from left signal, sample-by-sample.

## Usage

```
osc~ 440 → -~ ← osc~ 441
```

Useful for computing signal differences, differential signals, and phase cancellation effects.

## See Also

- [+~](/docs/objects/+~) - add signals
- [*~](/docs/objects/*~) - multiply signals
- [/~](/docs/objects//~) - divide signals
