Sawtooth ramp oscillator. Outputs a signal that ramps from 0 to 1 at the given frequency.

## Usage

```
440 → phasor~ → outputs sawtooth ramp 0→1 at 440 Hz
```

Useful as a phase source for waveshaping, FM synthesis, or driving other signal processors.
