# sheet

A spreadsheet-style data grid. Send a bang to output all cell values as a 2D array.

## Interactions

- **Double-click** a cell to edit it
- **Enter** to confirm, **Escape** to cancel, **Tab** to move to the next cell
- **Right-click** any cell to open the context menu:
  - Clear cell / Clear all cells
  - Add / remove row
  - Add / remove column
- **Drag** the right edge of a column header to resize it
- **Resize** the node by dragging its corners or edges
