# Keyboard Shortcuts

![Keyboard shortcuts](/content/images/patchies-shortcuts.png)

Patchies is designed to be keyboard-first so you can get in the flow. Go to "Help > Shortcuts" to see the full list of keyboard shortcuts.

## Essential Shortcuts

| Action | Shortcut |
|--------|----------|
| Create new object | `Enter` |
| Open object browser | `Ctrl/Cmd + O` |
| Save patch | `Ctrl/Cmd + S` |
| Save as | `Ctrl/Cmd + Shift + S` |
| New patch | `Ctrl/Cmd + N` |
| Command palette | `Ctrl/Cmd + K` |
| Toggle sidebar | `Ctrl/Cmd + B` |
| AI insert/edit | `Ctrl/Cmd + I` |
| Re-run code | `Shift + Enter` |
| Delete object | `Delete` or `Backspace` |
| Copy/Paste | `Ctrl/Cmd + C/V` |
