# Manage Saves

![Saves sidebar](/content/images/saves-sidebar.webp)

The Saves panel in the sidebar lets you manage your locally saved patches.

## Opening the Saves Panel

Use `Ctrl/Cmd + B > Saves` to show the saves panel.

## Organization

- Saves are organized into folders and can be filtered by name
- Hover over a save to load, rename and delete patches
- Right click on a save or folder to show a context menu

### Folder Actions

- Create subfolder
- Move
- Rename
- Delete

### Save Actions

- Load
- Share link
- Export as JSON
- Move
- Rename
- Delete

## Bottom Bar

Use the bottom bar to:
- Create a new folder
- Save patch as a new save
- Import patch from JSON files

## Keyboard Shortcuts

| Action | Shortcut |
|--------|----------|
| Save patch | `Ctrl/Cmd + S` |
| Save as | `Ctrl/Cmd + Shift + S` |
| New patch | `Ctrl/Cmd + N` |
