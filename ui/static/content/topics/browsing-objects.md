# Browsing Objects

![Object browser modal](/content/images/object-browser.webp)

- Use `Ctrl/Cmd + O` or the add button on the bottom right to open the object browser: a searchable, categorized view of objects and presets in Patchies
- Objects and presets are grouped by [packs](/docs/manage-packs), with searchable names and descriptions
- Click to insert an object or preset -- pick one at random and see what it can do
- Presets are pre-made objects that helps you get started quickly
  - You can [manage your own presets](/docs/manage-presets) in the sidebar: `Ctrl/Cmd + B > Presets`. Drag them into your canvas. You can make your own presets too
- Use the [packs panel](/docs/manage-packs) to add object and preset packs. If you search for disabled objects, a suggestion to enable relevant packs will also be shown
